# Phase 1 spine dependencies (Python 3.12, CPU-only).
#
# torch/torchaudio MUST come from the PyTorch CPU index, NOT PyPI (which pulls a
# multi-GB CUDA build). Install them FIRST, then the rest:
#
#   pip install torch==2.5.1 torchaudio==2.5.1 --index-url https://download.pytorch.org/whl/cpu
#   pip install -c constraints.txt -r requirements.txt   # constraints pins torch==2.5.1
#
# whisperx is pinned to 3.4.2 (NOT the plan's 3.1.5): 3.1.5 -> faster-whisper 1.0.1
# -> av==11.*, which has NO Python-3.12 wheel and forces a source build needing
# system pkg-config + libav dev headers. 3.4.2 is the newest whisperx that still
# allows torch>=2.5.1 (3.5.0+ jump to torch>=2.7.1 / ~=2.8.0 and would replace the
# CPU build). Its closure (faster-whisper 1.2.1, ctranslate2 4.4.0, av 17.1.0) all
# ship prebuilt cp312 wheels -> no source build. WhisperX word timing comes from
# the wav2vec2 align model, so caption quality is unchanged by the version bump.
